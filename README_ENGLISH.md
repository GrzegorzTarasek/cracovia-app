Cracovia App

An interactive application developed using Streamlit technology, serving as an analytical tool prepared for an internship at Cracovia football club.

Online application:
https://cracovia-app-necbudkx9z9eugjiihxyan.streamlit.app

The objective was to create an application enabling:
- analysis of data stored in an SQL database
- real-time filtering and exploration of information
- presentation of data in the form of tables and visualizations
- secure sharing of analytical results

The application combines the data processing layer (backend) with a simple and clear user interface.

Technologies used:
- Python
- Streamlit
- SQL
- Pandas
- Matplotlib / Plotly

Data protection (GDPR)

Due to personal data protection regulations (GDPR), surnames in the application are displayed only in the form of initials.
