Cracovia App
Interaktywna aplikacja stworzona w technologii Streamlit, pełniąca rolę narzędzia analitycznego przygotowanego na potrzeby praktyk w klubie Cracovia.

Aplikacja online:
https://cracovia-app-necbudkx9z9eugjiihxyan.streamlit.app

Celem było stworzenie aplikacji umożliwiającej:
- analizę danych zgromadzonych w bazie SQL
- filtrowanie i eksplorację informacji w czasie rzeczywistym
- prezentację danych w formie tabel i wizualizacji
- bezpieczne udostępnianie wyników analiz

Aplikacja łączy warstwę przetwarzania danych (backend) z prostym i czytelnym interfejsem użytkownika.

Wykorzystane technologie:
- Python
- Streamlit
- SQL
- Pandas
- Matplotlib / Plotly

Ochrona danych (RODO)
Ze względu na przepisy dotyczące ochrony danych osobowych (RODO), nazwiska w aplikacji prezentowane są wyłącznie w formie inicjałów.
